# talendComponent

This module includes add-ons required by talend etl.

* tPostgresSP allow to run a una store procedure on postgres 

## Requirements 

* Server Linux debian-like
* Java 1.7+
* Dog 2.5+
* Postgres 9+
* Talend 5.3

## Development

Import add-ons in talend Settings -> Talend -> Components
